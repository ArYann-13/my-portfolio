## ✨ Features

- **Sleek UI** – Built with **React.js** and **Tailwind CSS** for a modern look.
- **Dark/Light Mode** – Seamless theme toggling.
- **Project Showcase** – Interactive cards with smooth hover animations.
- **Contact Form** – Direct email integration via **Resend.com**.
- **Responsive Design** – Mobile‑first, fully adaptive.
- **Micro‑Animations** – Framer Motion powered effects and tooltips.

---

## 🛠️ Tech Stack

### Frontend
- **Framework:** React.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **HTTP Client:** Axios
- **Notifications:** React Toastify

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Email Service:** Resend
- **Middleware:** CORS

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/ArYann-13/my-portfolio.git
cd my-portfolio

# Frontend setup
cd frontend
npm install
npm start

# Backend setup (in a new terminal)
cd ../backend
npm install
node server.js
```

> ⚠️ **Note:** Create a `.env` file in `backend/` with your and add you Api from resend.com
